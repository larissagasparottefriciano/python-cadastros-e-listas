# python-cadastros-e-listas

Repositório de exercícios práticos em Python, desenvolvidos durante os estudos do curso de Análise e Desenvolvimento de Sistemas (ADS).

## Conteúdo

- Manipulação de listas
- Laços de repetição `for` e `while`
- Cadastros interativos via `input()` com validação de dados
- Uso de `break` e `continue` para controle de fluxo

## Objetivo

Praticar lógica de programação e estruturas de dados fundamentais em Python através de pequenos programas de cadastro (jogadores, sabores, produtos, etc).
